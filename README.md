## :books: 使用方法

### 源码运行(推荐)
1. 提前准备： node pnpm ts-node
2. `git clone https://github.com/s744007195/chaoxing.git` 项目至本地
3. `cd chaoxing`
4. `pnpm install`
5. `修改index.ts 账号密码配置`
5. `ts-node index.ts` 运行程序
